# 키워드 조사 백로그

발행 측정 사이클(~2주) 동안 다음 라운드 조사·큐레이션을 병렬 진행하기 위한 추적 문서. 측정 결과가 나오면 우선순위 재배열.

## 타겟 페르소나

**40+ 건강 관심층 (한국어 네이버 검색 사용자)**. 자세한 기준 — `docs/content-strategy.md` 의 "타겟 페르소나" 섹션 참조.

이 페르소나에 맞지 않는 키워드는 후보 단계에서 거절.

## 진행 상태 분류

- 🟢 **진행 중** — 데이터 받아서 큐레이션·자료조사 단계
- 🟡 **측정 대기** — 발행했고 결과 나오면 다음 발행 결정
- 🔵 **후보** — 데이터 받기 전 기획 단계
- ⚪ **보류** — 측정 결과 의존, 페르소나 부합 재검토 필요
- ❌ **거절** — 안티 패턴 / 페르소나 불일치 / 폐기 가설

## 진화 룰

- 측정 사이클 종료 시 결과를 반영해 우선순위 재배열
- 새 카테고리 페르소나 적합도 평가 → 후보·보류·거절 이동
- 한 키워드가 발행 후 완료되면 이 문서에서 제거(`experiments/` 로 이관)
- 다음 조사 토픽이 정해지면 🔵 후보 → 🟢 진행 중으로 이동
