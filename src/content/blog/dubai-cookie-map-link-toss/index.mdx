## 📱 토스 두쫀쿠 맵 하는법 (단계별)

두쫀쿠 맵 토스에서 확인하는 방법은 간단합니다.

**Step 1.** 토스 앱 실행

**Step 2.** 상단 검색창에 **'두쫀쿠 맵'** 또는 **'두쫀쿠'** 입력

**Step 3.** 검색 결과에서 **'두쫀쿠맵'** 미니앱 선택

**Step 4.** 지도에서 내 주변 매장과 실시간 재고 확인
- 🟢 초록색: 재고 여유
- 🟡 노란색: 재고 부족
- 🔴 빨간색: 품절

**Step 5.** 매장 선택하면 상세 정보 확인 가능
- 매장 위치, 영업시간
- 현재 재고 수량
- 마지막 업데이트 시간

### 💡 토스 두쫀쿠 맵 핵심 기능

- **실시간 재고 확인**: 매장 사장님이 직접 업데이트
- **위치 기반 검색**: 내 주변 매장 자동 표시
- **관심 매장 설정**: 재입고 알림 받기 가능
- **페이스페이 연동**: 토스 페이스페이로 바로 결제

### Q. 토스 두쫀쿠 맵과 웹사이트 두쫀쿠 맵, 뭐가 다른가요?

**같은 개발자가 만든 서비스**입니다. 기능은 거의 동일하고, 접근 방식만 다릅니다.

**두쫀쿠 맵 토스 (미니앱)**
- 위치 기반 자동 검색
- 재입고 알림 기능
- 페이스페이 결제 연동
- 토스 앱 필요
- 토스 생태계 내 추가 기능 활용 가능

**웹사이트 (dubaicookiemap.com)**
- 앱 설치 불필요
- PC에서도 사용 가능
- 누구나 접속 가능
- 토스 앱 없어도 이용 가능

💡 **개발 배경**: 개인 개발자가 여자친구를 위해 웹사이트를 먼저 만들었고, 입소문이 나자 토스가 미니앱 입점을 제안했습니다. 토스는 이 서비스를 구매한 것이 아니라, 토스 앱 내에서도 사용할 수 있도록 협력한 것입니다.

### Q. 두쫀쿠 맵 등록 방법은?

**카페 사장님의 경우:**

### Q. 재고 정보는 정확한가요?

**정확도는 높은 편**이지만 100% 보장은 어렵습니다.
- 매장 사장님이 직접 업데이트
- 마지막 업데이트 시간 확인 필수
- 재고 10개 미만이면 전화 확인 권장

## 💡 두쫀쿠 재고 확인 꿀팁

### 헛걸음 줄이는 방법

**1. 오픈 시간 직후 확인**
- 오전 오픈 직후가 재고 가장 많음
- 보통 11시~12시 사이

**2. 재고 여유 매장 선택**
- 🟢 초록색 표시 매장 우선 방문
- 재고 50개 이상인 곳이 안전

**3. 평일 오후 노려보기**
- 주말/공휴일보다 평일이 덜 복잡
- 오후 2~4시도 의외로 재고 있는 경우 많음

**4. 여러 매장 체크**
- 한 매장만 보지 말고 2~3곳 확인
- 가는 길목에 있는 매장 모두 체크

**5. 전화 확인이 가장 확실**
- 가기 전 5분만 투자해서 전화
- "지금 두쫀쿠 재고 있나요?" 물어보기

### 재고 업데이트 시간 체크법

두쫀쿠 맵에서 매장 정보를 클릭하면 **'마지막 업데이트 시간'** 이 표시됩니다.

- ✅ 30분 이내: 신뢰도 높음
- ⚠️ 1~2시간 전: 전화 확인 권장
- ❌ 3시간 이상: 정보 신뢰도 낮음
