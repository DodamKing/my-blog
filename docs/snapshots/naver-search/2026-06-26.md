## 직전(2026-05-18) 대비 핵심 변화

> **해석 한계 (먼저 명시)**:
> - 측정 기간 불일치: 직전 16일 vs 이번 30일, 갭(05-19~05-26) 미포착. 전체 현황(클릭/노출 -81.6%/-84.6%)은 네이버가 30일 vs 30일로 자체 계산한 수치라 신뢰 가능하지만, 이걸 직전 스냅샷의 16일 절대수치와 나란히 놓고 산술하면 기간 보정이 안 된 비교가 된다.
> - 페이지 TOP30 중 11~30위 데이터 없음 (캡처 미확보).
> - "지난달 대비"의 비교 대상 30일 구간이 정확히 언제인지 네이버 UI 기준이라 우리가 재현 불가.
> - 독립 비판 리뷰를 거쳐 아래 결론들의 과단정 표현을 1차 수정함.

### 1. redill 붕괴 — 트렌드 소멸 아니라 편집 사고의 지연 반영 (정정)
- redill-zero-purchase-guide: 클릭 827→23(-97%), 노출 38,354→608.
- **원인 정정**: `docs/content-strategy.md`에 이미 기록된 사건 — 05-18(827클릭, 편집 직전 피크) ~ 05-24 사이 "잘 나가는 글"의 이미지 교체 + 색인 재요청으로 점수가 떨어진 것 — 의 손상이 시간을 두고 누적되어 이번 스냅샷(23클릭)에서 끝까지 드러난 것이다. 트렌드 소멸이 아니다.

### 2. 단일 글 의존도 — 방향성은 명확, 정량 비교는 보류
- 직전: redill-zero-purchase-guide가 TOP30 페이지 클릭 합(1,491)의 55%.
- 이번: 1위 페이지(dyeing-shampoo)가 전체 사이트 클릭(270)의 11.9%. TOP10 합쳐도 62%.
- 두 비율은 분모 정의가 다르다(TOP30 합 vs 전체 사이트) — 직접 수치 비교는 불가. 다만 redill 자체 클릭이 827→23으로 무너진 것은 명백한 사실이라, "쏠림이 줄었다"는 방향성 자체는 견고하다. **단, 이건 의도적 분산 전략의 성과가 아니라 우리 편집 사고(이미지 교체+색인 재요청)로 인한 점수 하락의 부산물** — 자축할 일이 아니라 "분산 자산 부재 리스크가 자해성 편집 사고로 우연히 해소됐다"로 기록한다.

### 3. 사이트 전체 대폭락의 원인 — redill이 주요인으로 보이나 정량적 기여도는 미산정
- redill-zero-purchase-guide 한 페이지의 노출 붕괴(38,354→608, 거의 전멸 — 편집 사고로 인한 점수 하락, 트렌드 소멸 아님)가 강하게 시사하는 바는 있지만, 직전 스냅샷에 "전체 사이트 노출" 총량이 기록되어 있지 않아 "전체 감소분의 몇 %를 redill이 차지하는가"는 계산할 수 없다. **"거의 전부 redill 탓"이라는 정량적 단정은 데이터 부족으로 보류** — "주요인으로 보임" 수준까지만 채택.
- TOP10 페이지 중 redill 외 글들(dyeing-shampoo, dymatize, laptop-screen, optimum 등)은 절대 클릭 기준으로 안정적이거나 직전과 비슷한 수준 — 다른 구조적 악화 신호는 보이지 않는다(다만 11~30위 데이터가 없어 완전한 확인은 아님).

### 4. 최근 며칠 노출 급상승 — 발행 주기와 연결할 근거 없음
- 차트상 06-19경 노출 ~170 → 06-26 ~580으로 상승. 단일 차트의 일별 변동이라 원인 분리가 불가능하다. **"동시발행 여러 편 vs 분산발행"이 점수에 영향을 준다는 가설은 이 데이터로 뒷받침되지 않는다** (사용자 질문에 대한 답변, 본문 채팅 참조).
