발행 측정 사이클(~2주) 동안 다음 라운드 조사·큐레이션을 병렬 진행하기 위한 추적 문서. 측정 결과가 나오면 우선순위 재배열.

## 타겟 페르소나

**40+ 건강 관심층 (한국어 네이버 검색 사용자)**. 자세한 기준 — `docs/content-strategy.md` 의 "타겟 페르소나" 섹션 참조.

이 페르소나에 맞지 않는 키워드는 후보 단계에서 거절.

## 진행 상태 분류

- 🟢 **진행 중** — 데이터 받아서 큐레이션·자료조사 단계
- 🟡 **측정 대기** — 발행했고 결과 나오면 다음 발행 결정
- 🔵 **후보** — 데이터 받기 전 기획 단계
- ⚪ **보류** — 측정 결과 의존, 페르소나 부합 재검토 필요
- ❌ **거절** — 안티 패턴 / 페르소나 불일치 / 폐기 가설

데이터 파일: `data/keywords/back-pain-2026-05-26.json` (95 키워드). 메인 + 독립 심사 병렬 큐레이션 결과 **교집합 1편(`허리 디스크 터지면`)** 만 시도 가치였고, 그것도 "도박" 수준. 사용자가 폐기 결정 (이유: 허리 영역 의료·한방 권위 점유율 압도적, 1편 도박을 위해 도메인 신뢰도 희석 위험).

| 거절 군 | 사유 |
|---|---|
| 의류 수선 (바지/청바지/슬랙스/치마/레이어드/수선) — 18% | 40+ 건강 페르소나 이탈 |
| 의료기관 점유 (병원/도수치료/주사/수술/견인치료/내시경 감압술) | 외부 도메인 진입 불가 |

| 의료용 보호대·견인기·교정기 — 어필리에이트 농장 + 쇼핑몰 점유 | 진입 슬롯 0 + 안티 패턴(추천 TOP N) 위반 |
| 낚시 구명조끼 (다이와/구명벨트) | 페르소나 이탈 |
| 자기복제 — 허리 삐끗·담·허리베개 | `lower-back-strain-vs-cramp`, `jeong-seon-geun-back-pillow` 이미 점유 |
