## 배경

2026-05-02 측정에서 `redill-zero-purchase-guide` 단일 글이 전체 클릭의 약 61% 점유 → **트래픽 분산 자산 절실**.

동시에 새 승리 패턴 4 (단일 답변형, 천국의계단 효과 몇분 50% / 레딜 쿠팡 53.8% CTR) + 5 (불안감 진단형, 레딜 사기 9% / albumin 부작용 6.0% / 비타민C 메가도스 6.3%) 가 `content-strategy.md`에 박혔으나 사례 검증 강도가 아직 약함.

본 실험은 신규 패턴이 **다른 토픽(눈)에서도 동일하게 작동하는지** 표본 2편 동반 발행으로 검증한다.

## 검증할 가설

### H1 — 단일 답변형 × 자가진단 결합 (강함)
"단일 답변형(정량·짧은 답)" 패턴이 광고 블로그 회피 영역(자가진단·불안감)과 결합 시 외부 도메인 → 네이버 상위 진입이 가능하다.

| 판정 | 조건 |
|---|---|
| ✅ 합격 | 두 글 모두 발행 2주 시점 CTR ≥ 6% |
| 🟡 부분 합격 | 한 글만 ≥ 6% |
| ❌ 불합격 | 둘 다 < 6% |

### H2 — 불안감 진단형 보편성 (보조)
"불안감 진단형(부작용·위험·이상신호)" 패턴이 영양제·신제품 영역(기존 검증)을 넘어 **신체 자가진단 영역**에서도 작동한다.

판정 조건: 두 글이 안정적으로 인덱싱되고 노출 입력이 있을 것 (절대 노출 ≥ 100/2주)
