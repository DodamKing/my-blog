# 전기세 클러스터 발행 실험

**시작일**: 2026-05-24
**카테고리**: finance
**총 발행 예정**: 6편 (트렌드 1 + Evergreen 5)
**1차 목적**: 1페이지 진입 경험 누적 + finance 카테고리 권위 강화
**2차 목적**: redill 폭락 후 트래픽 회복

## 배경

2026-05-22 즈음 redill-zero-purchase-guide(사이트 노출 70% 점유)가 우리의 5/18 히어로 이미지 교체 + 색인 재요청 조합으로 노출 폭락. 일 노출 2,500 → 200. 학습: 잘 나가는 글은 건드리지 않는다 ([[feedback_dont_touch_winning_posts]]). 진행 룰 갱신: **1면 진입 경험 + 1면 글 다수 확보가 1차 목적, 체류는 2차.**

전기세 키워드는 2026-04-16 한국전력공사 시간대별 요금제 본격 도입 시점에 검색량 폭증. JSON 51개 큐레이션 + 메인/독립 비판 에이전트 교차검증 후 6편 추출.

## 가설

- 6편 중 3~4편이 1면 진입 가능 (메인 에이전트 SERP 실측 기준)
- 트렌드 1편(`가정용 전기세 개편`)은 단기 폭발 가능, 6개월 후 검색량 50%+ 빠짐
- Evergreen 5편은 1면 진입 시 안정 자산화 (매년 신규 출산·다자녀·이사 발생)
- finance 카테고리 도메인 권위 약함(pension-savings 글 한 달 반 노출 미진입)이 부정 변수

## 발행 로드맵

### Phase 1 — 트렌드 (즉시, 검색량 빠지기 전)

| 항목 | 값 |
|---|---|
| 슬러그(잠정) | `home-electricity-rate-reform` |
| 메인 키워드 | 가정용 전기세 개편 (1,950 검색 / 문서 919 / 경쟁률 0.47) |
| 차별화 앵글 | "가정용은 시간대 요금제 미적용" (산업용·일반용·전기차 충전기만 적용) |
| 보조 키워드 | 전기세 개편 (23,780) — 본문에 자연스럽게 포함시켜 2~3페이지 보조 노출 노림 |
| 검색 의도 | "전기료 시간대 개편됐다는데 우리 집은 어떻게 되는 거야?" |
| SERP 실측 | 개인블로그 3 / 미디어 5 (점유 강함 — 차별화 앵글이 1면 진입 핵심) |
| 1면 진입 확률 | 40~60% (1~3위는 15~25%, 8~10위 현실적) |

## 검증/기각 기준

- **6편 중 1면 진입 4편+** → 전략 성공. finance 카테고리 추가 발굴 확정
- **6편 중 1면 진입 1편 이하** → finance 도메인 권위 약함 가설 강화. 다른 카테고리(health 비-의료 / tech 트러블슈팅)로 회전
- **트렌드 글이 발행 1주 내 일 클릭 100+** → 트렌드 키워드 단기 폭발 패턴 확정. 다음 정책 이슈에도 빠르게 1편 발행 룰 추가

## 측정 일정

- 발행 후 **1주, 2주, 1개월** 시점 노출/클릭 추적
- 1면 진입 확인: 네이버에서 메인 키워드 직접 검색 → 우리 글 순위 확인
- 다음 정기 스냅샷 리뷰일(`2026-06-01`)에 1차 결과 누적 기록
- **잘 나가는 글이 나오면 절대 건드리지 않는다** ([[feedback_dont_touch_winning_posts]])

## 검색 의도 + 배경 (글 작성 시 참조)

### 가정용 전기세 개편
- 2026-04-16부터 한국전력공사가 시간대별 요금제 본격 도입 (평일 18~21시 최고요금, 낮 시간 인하)
- 단 **가정용은 미적용** — 산업용·일반용·전기차 충전기만 적용
- 미디어 헤드라인은 "전기료 시간대 개편"으로 가정용 포함처럼 보이지만 실제론 미적용
- 사용자(가정용 사용자)들이 "내 전기세도 바뀌나? 시간대 신경 써야 하나? 누진제는 그대로?" 검색 폭증
- 이 미디어 헤드라인 vs 실제 정책의 미스매치가 우리 차별화 앵글

### 신생아 전기세 감면
- 출산 후 한전ON에서 신청. 월 30% 할인 (한도 있음). 자동 적용 안 됨 — 신청 필수
- 이사 시 재신청 누락 함정 (한전이 따로 알려주지 않음)
- 감면 소급 불가 (신청 시점부터만 적용)

### 다자녀 전기세 할인
- 자녀 3명 이상 가구 대상. 월 30% 할인, 한도 월 16,000원
- 자녀 2명 가구는 일부 지자체만 가능 — 이게 "전기세 다자녀 2명" 검색의 답
- 출산 감면과 중복 적용 가능 (별도 신청)

### 전기세 할증시간
- "할증시간" = "피크타임" — 산업용·일반용 평일 18~21시 최고요금
- 가정용은 누진제 그대로 유지 (할증시간 없음)
- 검색자는 "내 가정용 전기세에 할증시간 있나?"가 진짜 질문 → "가정용 미적용"이 답

## 폐기 조건 (이 실험을 멈출 시점)

- Phase 1~2 발행 후 1개월 내 1면 진입 0편 → 클러스터 전체 중단
- 트렌드 글이 발행 후 2주간 클릭 30 미만 → 미디어 점유율을 우리가 뚫지 못함을 확정. Phase 3, 4 재검토
