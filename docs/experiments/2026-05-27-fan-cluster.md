**시작일**: 2026-05-27
**카테고리**: other (가전)
**총 발행 예정**: 6편 (Evergreen 3 + 브랜드 시즌 3)
**1차 목적**: other 카테고리 1면 진입 글 확장 + 시즌 트래픽 흡수
**2차 목적**: "쉬움 라벨 함정" vs "1면 도메인 분포" 가중치 검증 — 데이터 라벨에 끌리지 않고 SERP 실측만으로 채택한 키워드의 진입률 측정

## 배경

2026-05-27 사용자가 던진 선풍기 JSON 95개 → 41개 추림 → 메인 큐레이션 + 비판 독립 교차검증. 전립선/에어컨 JSON 2회 거절 누적 후 첫 발행 시도.

세 키워드 카테고리 거절 후 사용자 직감("이 키워드가 꿀일 것 같아") + 데이터 라벨 매력도(루메나 opp 99.7, 참바디 문서 27) 미련 누적 상황에서, 메인 에이전트가 5편 / 독립 에이전트가 3~4편 권고로 충돌. 사용자가 SERP 스크린샷으로 직접 반박해 일부 거절 사유 재검토 → 6편으로 최종 확정.

**핵심 학습 (이 실험으로 검증할 가설)**:
- `competition_ratio` 0.x = 빈 영역 신호이지만 단독 결정자가 아님
- 1면 도메인 분포가 결정적: 공식·쇼핑·가격비교 100% 점유 = 함정, SEO팜·블로그 슬롯 1+ = 진입 가능
- 트렌드 단발성과 진입 가능성은 별개 차원으로 분리해서 평가

## 가설

- **Evergreen 3편(항공반입·버리는법·날개빼는법)**: 1면 진입률 80%+, 시즌 무관 안정 자산. albumin-side-effects-guide 같은 진단·정보형 안정 패턴 재현 기대
- **브랜드 3편(루메나·샤크·참바디)**: 시즌 폭발 가능. 단 1년 후 검색량 50% 이상 빠질 위험 감수 (특히 샤크·참바디는 신생 브랜드)
- **SERP 실측 함정 가설 검증**: 데이터에서 가장 빛난 키워드(루메나 opp 99.7 / 샤크 문서 3,483 / 참바디 문서 27)를 모두 시도 → 진입률 비교로 "신생 브랜드 1면 선점" 가설 검증
- **신생 vs 안정 브랜드 진입률 비교**: 샤크/참바디(신생) vs 루메나(안정) 진입률 차이로 "신생 브랜드 = 1면 권위 굳기 전 진입 기회" 가설 정량화

## 측정 지표

메모리 피드백 [feedback_no_self_data_as_evidence](MEMORY.md), 측정 원칙 `docs/content-strategy.md` 적용. **절대 클릭이 1차, CTR은 2차.** 단기 CTR 변동으로 패턴 강약 판정 금지.

### 1차 지표 — 절대 클릭 수
- **소스**: 네이버 서치 어드바이저 페이지별 클릭
- **측정 시점**: 발행 후 +1주, +2주, +4주, +8주, +12주(시즌 종료 = 10월 초)
- **기준**:
  - 발행 +2주 일 클릭 10+ = 인덱싱 정상
  - +4주 누적 클릭 50+ = 1면 진입 가능성 신호
  - +8주 누적 클릭 200+ = 안정 자산 후보

### 2차 지표 — 노출·CTR·평균 순위
- **노출**: 검색 노출 수 (페이지별)
- **CTR**: 클릭/노출 비율. 패턴 강약 판정 도구가 아니라 **글 제목·메타 디스크립션 개선 신호로만 사용**
- **평균 순위**: 서치 어드바이저 자동 집계 — PC/모바일 혼합이라 부정확. **참고 보조 지표**

### 3차 지표 — 체류·이탈
- **소스**: Google Analytics
- **체류시간**: 페이지별 평균 — 1면 진입 후 SERP 클릭 → 체류 짧으면 직행이탈로 순위 떨어짐
- **이탈률**: 단일 페이지에서 나가는 비율

### 위험 지표 (redill 패턴 알람)
- **단일 글 집중도**: 6편 중 단일 글이 클러스터 클릭 60%+ 점유 → 분산 다양화 신호. 다음 발행 자제
- **시즌 종료 후 검색량 감소율**: 10월 시점 vs 7~8월 피크 비교. 50%+ 빠지면 트렌드 단발성 자산으로 재분류

### 1면 진입 확인 방법
- **수동 검증**: 네이버에서 메인 키워드 직접 검색 → PC + 모바일 둘 다 우리 글 순위 확인
- **자동 보조**: 서치 어드바이저 평균 순위 (정확도 낮음, 참고용)
- **주기**: 매주 1회 6편 일괄 확인 (월요일 기준)

## 검색 의도 + 배경 (각 글 작성 시 참조)

### 1. 선풍기 버리는법
- 1m 기준 (이하 무상수거, 이상 대형폐기물 스티커)
- 무상방문수거 1599-0903 (한국전자제품자원순환공제조합)
- 지자체별 스티커 가격 (대략 2,000~5,000원, 표 작성 필요)
- 첫 줄 결론: "1m 이하 = 무상, 이상 = 대형폐기물 스티커"
- 차별화: SEO팜 4글이 결론을 흐리게 풀어쓰므로 우리는 첫 줄 단답 + 사이트 직접 링크

### 2. 손선풍기 기내반입·위탁수화물 통합
- 위탁수화물 = **금지** (리튬 배터리 화재 위험)
- 기내반입 = **허용** (100Wh 이하 5개, 160Wh는 항공사 승인)
- 2026-04-20 한국항공·아시아나 보조배터리 정책 변경 반영 (본문에 "2026" 박지 말 것)
- 국가별 분기: 일본·동남아 항공사별 차이 H2 섹션
- 첫 줄 결론 + 항공사별 표 + Wh 계산법

### 3. 선풍기 날개 빼는법
- 브랜드별 분해 방향 (시계 반대, 풀림 표시 ▶ 확인)
- 신일·한일·루메나·프롬비 분기
- 단답형 첫 줄 + 모델별 사진 설명
- 차별화: 문서 253건 = 거의 빈 영역. 우리 글이 표준 답안 만들 기회
