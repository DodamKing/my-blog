키보드가 갑자기 고장 났을 때, 화면이 갑자기 확대됐을 때, 급하게 음성을 녹음해야 할 때.
별도 프로그램을 설치하지 않아도 윈도우에는 이미 해결책이 내장되어 있습니다.

문제는 이 기능들이 어디에 있는지, 어떻게 켜고 끄는지 모르는 분들이 많다는 것입니다.

이 글에서는 다음 네 가지 기본 내장 기능을 다룹니다.

- **가상키보드** — 물리 키보드 없이 입력하는 방법
- **돋보기** — 갑자기 확대된 화면 되돌리기
- **녹음기** — 설치 없이 바로 음성 녹음
- **달력** — 별도 앱 없이 일정 관리

## 윈도우 가상키보드(화상키보드) 여는 방법

물리 키보드가 고장 나거나, 태블릿 모드에서 입력이 필요할 때 가상키보드를 사용할 수 있습니다.
윈도우에는 **화상 키보드**와 **터치 키보드** 두 가지가 있는데, 여는 방법을 상황별로 정리했습니다.

### 상황별 추천 방법

| 상황 | 추천 방법 | 단축키/경로 |
|------|----------|------------|
| 키보드 일부만 고장 | 단축키 | **Ctrl + Win + O** |
| 키보드 완전 먹통 | 설정에서 열기 | 설정 > 접근성 > 키보드 |
| 로그인 화면에서 | 접근성 아이콘 | 로그인 화면 우측 하단 |
| 터치스크린 사용 | 작업 표시줄 | 작업 표시줄 우클릭 > 터치 키보드 |
| 빠르게 한 번만 | 실행 명령어 | Win + R > `osk` 입력 |

### 방법 1: 단축키 (가장 빠름)

**Ctrl + Win + O**를 누르면 화상 키보드가 바로 나타납니다.

키보드가 완전히 먹통인 상태라면 이 방법은 쓸 수 없으니, 아래 마우스만으로 여는 방법을 확인하세요.

### 방법 2: 설정에서 열기

1. **설정**(Win + I) 열기
2. **접근성** 선택
3. **키보드** 클릭
4. **화상 키보드** 토글을 켜기

### 방법 3: 실행 명령어

1. **Win + R** 눌러 실행 창 열기
2. `osk` 입력 후 엔터

### 방법 4: 터치 키보드 표시 (작업 표시줄)

1. 작업 표시줄 빈 공간을 **마우스 오른쪽 클릭**
2. **터치 키보드 버튼 표시** 선택
3. 작업 표시줄에 나타난 키보드 아이콘 클릭

### 방법 5: 로그인 화면에서 열기

비밀번호 입력이 필요한데 키보드가 안 될 때 유용합니다.

1. 로그인 화면 **오른쪽 하단** 접근성 아이콘(사람 모양) 클릭
2. **화상 키보드** 선택

### 화상 키보드 vs 터치 키보드, 뭐가 다를까?

| 구분 | 화상 키보드 | 터치 키보드 |
|------|------------|------------|
| 파일명 | osk.exe | TabTip.exe |
| 용도 | 접근성 (마우스 클릭 입력) | 터치스크린 입력 |
| 단축키 | Ctrl + Win + O | 작업 표시줄 아이콘 |
| 크기 조절 | 자유롭게 가능 | 하단 고정 |
| 추천 상황 | 키보드 고장, 마우스만 사용 | 태블릿 모드, 터치 입력 |

둘 다 물리 키보드 없이 입력할 수 있다는 점은 같지만, **마우스로 클릭해서 입력**하려면 화상 키보드가, **터치로 입력**하려면 터치 키보드가 더 편합니다.

### 가상키보드가 안 열릴 때

화상 키보드나 터치 키보드가 실행되지 않으면 관련 서비스가 꺼져 있을 수 있습니다.

1. **Win + R** > `services.msc` 입력
2. **Touch Keyboard and Handwriting Panel Service** 찾기
3. 상태가 "중지됨"이면 **마우스 오른쪽 클릭** > **시작**
4. 시작 유형을 **자동**으로 변경

## 가상키보드가 자동으로 뜰 때 끄는 법

원하지 않는데 가상키보드가 계속 나타나는 경우도 있습니다. 특히 입력란을 클릭할 때마다 터치 키보드가 뜬다면 아래 방법으로 끌 수 있습니다.

### 방법 1: 설정에서 끄기

1. **설정** > **시간 및 언어** > **입력**
2. **터치 키보드** 섹션에서 "키보드가 연결되지 않은 경우 터치 키보드 표시" 옵션을 **끄기**

### 방법 2: 서비스 비활성화 (완전 차단)

1. **Win + R** > `services.msc` 입력
2. **Touch Keyboard and Handwriting Panel Service** 찾기
3. 마우스 오른쪽 클릭 > **속성**
4. **서비스 중지** 클릭
5. 시작 유형을 **사용 안 함**으로 변경

> 이 서비스를 비활성화하면 터치 키보드와 손글씨 입력 기능이 모두 꺼집니다. 터치스크린을 사용하는 경우에는 방법 1만 적용하세요.

### 방법 3: 로그인 시 화상 키보드 자동 실행 끄기

부팅할 때마다 화상 키보드가 뜨는 경우입니다.

1. **제어판** > **접근성 센터** 열기
2. **로그인 설정 변경** 클릭
3. "화상 키보드 사용" 체크 해제
4. 적용

## 윈도우 돋보기 끄기 & 비활성화

화면이 갑자기 확대되어 당황한 적 있으신가요? 실수로 **Win + +(플러스)** 키를 눌러 돋보기가 실행된 것일 수 있습니다.

### 돋보기 끄기: Win + Esc

가장 빠른 방법입니다. **Win + Esc**를 누르면 즉시 돋보기가 종료됩니다.

### 돋보기 기본 단축키 정리

| 기능 | 단축키 |
|------|--------|
| 돋보기 켜기 | Win + +(플러스) |
| 돋보기 끄기 | Win + Esc |
| 확대 | Win + +(플러스) |
| 축소 | Win + -(마이너스) |
| 돋보기 설정 열기 | Win + Ctrl + M |

### 돋보기 자동 실행 비활성화

매번 로그인할 때마다 돋보기가 켜진다면 설정을 바꿔야 합니다.

1. **설정** > **접근성** > **돋보기**
2. "로그인 후 돋보기 시작" 옵션을 **끄기**
3. "모든 사용자에 대해 로그인 전에 돋보기 시작" 도 **끄기**

그래도 해결되지 않으면 **작업 관리자**(Ctrl + Shift + Esc) > **시작 앱** 탭에서 Magnifier를 찾아 비활성화하세요.

### 돋보기 3가지 모드

돋보기를 끄지 않고 활용하고 싶다면, 모드를 알아두면 유용합니다.

| 모드 | 설명 | 전환 단축키 |
|------|------|------------|
| 전체 화면 | 화면 전체를 확대 | Ctrl + Alt + F |
| 렌즈 | 마우스 주변만 확대 | Ctrl + Alt + L |
| 도킹 | 화면 상단에 확대 영역 고정 | Ctrl + Alt + D |

어르신이나 저시력 사용자에게는 돋보기가 매우 유용한 도구입니다. 끄는 법만 알면 실수로 켜졌을 때도 당황하지 않을 수 있습니다.

## 윈도우 녹음기 사용 방법

급하게 음성을 녹음해야 하는데 별도 프로그램이 없다면, 윈도우 기본 녹음기를 활용하세요. 설치 없이 바로 사용할 수 있습니다.

> 윈도우 10에서는 "음성 녹음기(Voice Recorder)", 윈도우 11에서는 "녹음기(Sound Recorder)"라는 이름입니다.

### 녹음기 앱 실행

1. **시작 메뉴**에서 "녹음기" 검색
2. 앱 클릭하여 실행

### 녹음 시작부터 저장까지

1. **빨간 원 버튼** 클릭 (또는 **Ctrl + R**)으로 녹음 시작
2. 녹음 중 **일시정지** 버튼으로 잠시 멈출 수 있음
3. **정지** 버튼을 누르면 녹음이 끝나고 자동 저장

### 녹음기 주요 단축키

| 기능 | 단축키 |
|------|--------|
| 녹음 시작 | Ctrl + R |
| 일시정지/재개 | Alt + P |
| 녹음 중지 | Esc |
| 마커 추가 | Ctrl + M |
| 트리밍(자르기) | Ctrl + T |

### 녹음 파일은 어디에 저장될까?

녹음이 끝나면 자동으로 **문서 > 녹음 파일** 폴더에 저장됩니다.
파일 형식은 윈도우 11 기준 m4a가 기본이며, 설정에서 **mp3, wma, flac, wav** 등으로 변경할 수 있습니다.

### 마이크가 인식되지 않을 때

녹음 버튼이 비활성화되어 있다면 마이크 권한을 확인하세요.

1. **설정** > **개인 정보 및 보안** > **마이크**
2. "앱이 마이크에 액세스하도록 허용" 을 **켜기**
3. 앱 목록에서 "녹음기"도 **켜기**

### 시스템 소리(컴퓨터 소리)도 녹음할 수 있을까?

기본 녹음기는 **마이크 입력만** 녹음합니다. 컴퓨터에서 나오는 소리(유튜브, 음악 등)를 녹음하려면 **스테레오 믹스**를 활성화해야 합니다.

1. **제어판** > **소리** > **녹음** 탭 열기
2. 빈 공간에 마우스 오른쪽 클릭 > "사용 안 함 장치 표시" 체크
3. **스테레오 믹스**가 나타나면 마우스 오른쪽 클릭 > **사용**
4. 스테레오 믹스를 **기본 장치로 설정**
5. 녹음기 앱에서 녹음하면 시스템 소리가 녹음됨

> 일부 노트북에서는 스테레오 믹스가 지원되지 않을 수 있습니다. 이 경우 별도 녹음 프로그램이 필요합니다.

## 윈도우 달력 활용 팁

윈도우에 기본으로 들어 있는 달력(일정) 앱을 활용하면 별도 프로그램 없이도 간단한 일정 관리가 가능합니다.

### 가장 빠르게 달력 보는 법

작업 표시줄 오른쪽 하단의 **시계를 클릭**하면 달력이 바로 나타납니다.
여기서 날짜를 클릭하면 간단한 일정도 추가할 수 있습니다.

### 일정 앱으로 본격 일정 관리

더 자세한 일정 관리가 필요하면 "일정" 앱을 사용하세요.

1. 시작 메뉴에서 "일정" 검색
2. 앱 실행 후 **일/주/월** 단위로 보기 전환 가능
3. 원하는 날짜 클릭 > 일정 제목, 시간, 장소 입력
4. **알림 설정**으로 일정 전에 미리 알림 받기

### 구글 캘린더, 아웃룩 연동하기

윈도우 일정 앱에 외부 캘린더를 연동하면 여러 플랫폼의 일정을 한 곳에서 볼 수 있습니다.

1. 일정 앱 > **설정(톱니바퀴)** 클릭
2. **계정 관리** > **계정 추가**
3. Google, Outlook, iCloud 중 선택
4. 로그인하면 자동 동기화

### 달력 앱 활용 팁 정리

| 기능 | 방법 |
|------|------|
| 빠르게 달력 보기 | 작업 표시줄 시계 클릭 |
| 일정 추가 | 날짜 클릭 > 일정 입력 |
| 외부 캘린더 연동 | 설정 > 계정 추가 |
| 보기 전환 | 일/주/월 단위 전환 |
| 알림 | 일정별 알림 시간 설정 |
| PDF로 저장 | 인쇄 > PDF 프린터 선택 |

## FAQ

**Q. 가상키보드와 터치키보드는 같은 건가요?**

아닙니다. 화상 키보드(osk.exe)는 접근성 도구로 마우스 클릭 입력에 최적화되어 있고, 터치 키보드(TabTip.exe)는 터치스크린 입력용입니다. 기능은 비슷하지만 UI와 용도가 다릅니다.

**Q. 돋보기가 자꾸 자동으로 켜져요. 어떻게 하나요?**

설정 > 접근성 > 돋보기에서 "로그인 후 돋보기 시작" 옵션을 끄세요. 그래도 해결되지 않으면 작업 관리자 > 시작 앱에서 Magnifier를 찾아 비활성화하면 됩니다.

**Q. 녹음기로 유튜브 소리나 컴퓨터 소리를 녹음할 수 있나요?**

기본 녹음기는 마이크 입력만 녹음합니다. 시스템 소리를 녹음하려면 제어판 > 소리 > 녹음 탭에서 "스테레오 믹스"를 활성화한 뒤 기본 장치로 설정하세요. 일부 노트북에서는 스테레오 믹스가 지원되지 않을 수 있습니다.

**Q. 윈도우 달력에서 구글 캘린더 일정을 볼 수 있나요?**

네. 윈도우 일정 앱에서 설정 > 계정 추가 > Google을 선택하면 구글 캘린더와 동기화됩니다. 양쪽에서 추가한 일정이 모두 반영됩니다.

**Q. 이 기능들은 윈도우 10과 11 모두 사용할 수 있나요?**

네. 가상키보드, 돋보기, 녹음기, 달력 모두 윈도우 10과 11에서 사용할 수 있습니다. 다만 메뉴 경로나 앱 이름이 조금 다를 수 있습니다. 예를 들어 녹음기 앱은 윈도우 10에서 "음성 녹음기", 윈도우 11에서 "녹음기"라는 이름입니다.
