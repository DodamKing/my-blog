윈도우 디펜더가 특정 프로그램 설치를 차단하거나, 다른 백신을 쓰고 있어서 끄고 싶을 때가 있습니다. 혹은 PC 성능을 조금이라도 더 끌어올리고 싶어서 **윈도우 디펜더 끄기**를 검색하신 분도 계실 겁니다.

이 글에서는 **일시적으로 끄는 방법**, **그룹 정책으로 영구 비활성화하는 방법**, **레지스트리로 완전히 끄는 방법** 3가지를 상황별로 정리했습니다. 끄기 전에 알아야 할 주의사항과 대안도 함께 다루니 끝까지 읽어보시기 바랍니다.

### 3줄 요약

- **잠깐 끄기**: 설정에서 실시간 보호를 끄면 되지만, 재부팅하면 자동으로 다시 켜집니다.
- **영구 끄기 (Pro 이상)**: 그룹 정책 편집기로 완전히 비활성화할 수 있습니다.
- **영구 끄기 (Home)**: 레지스트리 편집으로 비활성화 가능하지만 신중하게 진행해야 합니다.

## 윈도우 디펜더란? 왜 끄려고 할까

**윈도우 디펜더(Microsoft Defender)** 는 Windows에 기본 내장된 보안 프로그램입니다. 실시간으로 바이러스, 랜섬웨어, 스파이웨어 등 악성 소프트웨어를 감지하고 차단합니다.

보안 평가 기관 AV-TEST에서 보호, 성능, 사용성 모든 항목에서 6점 만점을 받을 정도로 성능이 좋아졌지만, 그래도 끄고 싶은 상황이 있습니다.

### 디펜더를 끄려는 흔한 이유

| 상황 | 설명 |
|------|------|
| 프로그램 설치 차단 | 크랙, 키젠, 특정 개발 도구 등을 악성코드로 오진하여 차단 |
| 다른 백신 사용 중 | 카스퍼스키, 어베스트 등 별도 백신과 충돌 가능성 |
| PC 성능 최적화 | 실시간 검사가 CPU와 디스크를 점유하여 성능 저하 |
| 파일 삭제 문제 | 다운로드한 파일을 자동으로 삭제해버리는 경우 |

> 💡 **참고**: 다른 백신 프로그램을 설치하면 윈도우 디펜더는 대부분 자동으로 비활성화됩니다. 이 경우 별도로 끄지 않아도 됩니다.

## 설정에서 실시간 보호 일시 끄기

가장 간단한 방법입니다. 프로그램 설치처럼 잠깐 끄고 싶을 때 사용하세요.

### 따라하기

1. Windows 키 + I → 설정 열기
2. 개인 정보 및 보안 → Windows 보안 클릭
3. 바이러스 및 위협 방지 클릭
4. 바이러스 및 위협 방지 설정 아래 "설정 관리" 클릭
5. 실시간 보호 토글을 "끔"으로 변경
6. 사용자 계정 컨트롤 팝업에서 "예" 클릭

> **Windows 10 사용자**: 설정 → 업데이트 및 보안 → Windows 보안 → 바이러스 및 위협 방지 순서로 진입합니다.

### 알아두어야 할 점

- **일시적입니다.** 재부팅하거나 일정 시간이 지나면 자동으로 다시 켜집니다.
- Microsoft가 보안을 위해 이 설정이 꺼진 상태를 허용하지 않기 때문입니다.
- 프로그램 설치 같은 **단발성 작업**에 적합합니다.

⚠️ **주의**: 실시간 보호가 꺼진 상태에서는 인터넷 서핑이나 파일 다운로드를 최소화하세요.

## 그룹 정책으로 영구 비활성화하기

**Windows Pro, Enterprise, Education 에디션**에서 사용할 수 있는 방법입니다. Home 에디션에서는 그룹 정책 편집기를 사용할 수 없으므로, 다음 섹션의 레지스트리 방법을 참고하세요.

### 사전 준비: 변조 방지(Tamper Protection) 끄기

그룹 정책이나 레지스트리로 디펜더를 끄려면 **반드시 변조 방지를 먼저 꺼야** 합니다. 이 기능이 켜져 있으면 사용자가 수정한 설정이 자동으로 원래대로 돌아갑니다.

1. Windows 키 + I → 설정 열기
2. 개인 정보 및 보안 → Windows 보안
3. 바이러스 및 위협 방지 → 설정 관리
4. 아래로 스크롤하여 "변조 방지" 항목을 "끔"으로 변경
5. 사용자 계정 컨트롤 팝업에서 "예" 클릭

### 그룹 정책 편집기로 비활성화하기

1. Windows 키 + R → "gpedit.msc" 입력 후 확인
2. 왼쪽 트리에서 아래 경로로 이동: 컴퓨터 구성 → 관리 템플릿 → Windows 구성 요소 → Microsoft Defender 바이러스 백신
3. 오른쪽에서 "Microsoft Defender 바이러스 백신 끄기" 더블클릭
4. "사용"을 선택하고 확인 클릭
5. PC 재부팅

> 💡 **팁**: Windows 10에서는 "Microsoft Defender 바이러스 백신" 대신 "Windows Defender 바이러스 백신"으로 표시될 수 있습니다.

### 다시 켜는 방법

같은 경로에서 "Microsoft Defender 바이러스 백신 끄기" 항목을 **"구성되지 않음"** 으로 변경하고 재부팅하면 됩니다.

## 레지스트리로 완전히 비활성화하는 방법

**Windows Home 에디션** 사용자이거나 그룹 정책 편집기가 없는 경우 이 방법을 사용합니다.

⚠️ **경고**: 레지스트리 수정은 시스템에 영향을 줄 수 있습니다. 반드시 아래 주의사항을 읽고 진행하세요.

### 사전 준비

1. **변조 방지(Tamper Protection)를 먼저 끄세요** — 위 그룹 정책 섹션의 "사전 준비" 부분과 동일합니다.
2. **레지스트리 백업을 권장합니다** — 레지스트리 편집기에서 파일 → 내보내기로 현재 상태를 저장해두세요.

### 레지스트리 편집하기

1. Windows 키 + R → "regedit" 입력 후 확인
2. 사용자 계정 컨트롤 팝업에서 "예" 클릭
3. 아래 경로로 이동: `HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender`
4. 오른쪽 빈 공간에서 우클릭 → 새로 만들기 → DWORD(32비트) 값
5. 이름을 "DisableAntiSpyware"로 입력
6. 생성된 값을 더블클릭 → 값 데이터를 "1"로 변경 → 확인
7. PC 재부팅

### 이미 DisableAntiSpyware가 있는 경우

해당 값을 더블클릭해서 값 데이터가 **0**이면 **1**로 바꾸면 됩니다.

### 다시 켜는 방법

같은 경로에서 DisableAntiSpyware 값의 데이터를 **0**으로 변경하거나, 해당 값을 삭제하고 재부팅하면 됩니다.

### 윈도우 에디션별 방법 비교

| 항목 | 설정에서 끄기 | 그룹 정책 | 레지스트리 |
|------|:---:|:---:|:---:|
| 지속성 | 일시적 | 영구적 | 영구적 |
| Home 에디션 | O | X | O |
| Pro 이상 에디션 | O | O | O |
| 난이도 | 쉬움 | 보통 | 보통 |
| 되돌리기 | 자동 복구 | 쉬움 | 쉬움 |

## 디펜더를 꺼도 안전한가? 대안 백신 추천

### 끄면 어떤 위험이 있나

윈도우 디펜더를 끄면 PC가 외부 위협에 **무방비 상태**가 됩니다. 특히 아래 상황에서 위험합니다.

- **랜섬웨어 감염**: 파일이 암호화되어 복구 불가능
- **악성코드 설치**: 개인정보 유출, 원격 제어
- **피싱 사이트 접속**: 보호 기능 없이 악성 사이트에 노출

### 끄는 것보다 예외 설정을 권장

특정 프로그램이 차단되는 문제라면 **디펜더를 끄지 않고 예외를 추가**하는 것이 안전합니다.

1. 설정 → Windows 보안 → 바이러스 및 위협 방지
2. 설정 관리 → 아래로 스크롤
3. 제외 → "제외 추가 또는 제거" 클릭
4. 제외 추가 → 파일, 폴더, 파일 형식, 프로세스 중 선택
5. 차단되는 항목을 지정하여 예외 등록

이 방법이면 디펜더는 계속 작동하면서 **특정 파일이나 폴더만 검사에서 제외**됩니다.

### 그래도 꺼야 한다면: 대안 백신

디펜더를 끄고 다른 백신을 사용하려면 아래 옵션을 고려해보세요.

| 백신 프로그램 | 가격 | 특징 |
|------|------|------|
| Bitdefender | 무료/유료 | 가벼운 리소스 사용, 높은 탐지율 |
| Kaspersky Free | 무료 | 강력한 보호 성능, 러시아 기업 |
| Avast Free | 무료 | 사용하기 쉬운 인터페이스 |
| Malwarebytes | 무료/유료 | 기존 백신과 함께 사용 가능 (보조 백신) |

> 💡 **참고**: 별도의 백신 프로그램을 설치하면 윈도우 디펜더는 대부분 자동으로 비활성화됩니다. 이 경우 수동으로 끄는 과정이 필요 없습니다.

## FAQ

**Q. 윈도우 디펜더를 끄면 PC가 느려지지 않나요?**

오히려 반대입니다. 디펜더의 실시간 검사가 CPU와 디스크를 사용하기 때문에, 끄면 일부 작업에서 성능이 향상될 수 있습니다. 다만 보안 위협에 노출되므로 다른 백신 없이 장기간 사용하는 것은 권장하지 않습니다.

**Q. 디펜더를 껐다가 다시 켤 수 있나요?**

네, 모두 되돌릴 수 있습니다. 설정에서 끈 경우 재부팅하면 자동으로 켜지고, 그룹 정책이나 레지스트리로 끈 경우에도 해당 설정을 원래대로 변경하면 다시 활성화됩니다.

**Q. 윈도우 Home 에디션인데 gpedit.msc가 안 열려요**

윈도우 Home 에디션에는 그룹 정책 편집기가 기본 포함되어 있지 않습니다. 이 경우 **레지스트리 편집기(regedit)** 방법을 사용하세요. 위의 "레지스트리로 완전히 비활성화하는 방법" 섹션을 참고하시면 됩니다.

**Q. 디펜더를 끄지 않고 특정 파일만 허용할 수 있나요?**

네, **예외 설정**으로 가능합니다. 설정 → Windows 보안 → 바이러스 및 위협 방지 → 설정 관리 → 제외에서 특정 파일, 폴더, 파일 형식, 프로세스를 검사 대상에서 제외할 수 있습니다. 디펜더를 끄는 것보다 이 방법을 먼저 시도해보세요.

**Q. 변조 방지(Tamper Protection)를 안 끄고 레지스트리를 수정하면 어떻게 되나요?**

레지스트리 값을 변경해도 Windows가 자동으로 원래 설정으로 되돌립니다. 그래서 **반드시 변조 방지를 먼저 끈 상태에서** 그룹 정책이나 레지스트리 수정을 진행해야 합니다.
