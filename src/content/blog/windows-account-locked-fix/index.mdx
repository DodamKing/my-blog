윈도우에 로그인하려는데 **"이 계정은 잠겨 있습니다"** 메시지가 뜨면 당황스럽습니다. **윈도우 계정 잠김**은 비밀번호를 여러 번 틀리거나, Windows 업데이트 이후 갑자기 발생하기도 합니다.

이 글에서는 윈도우 계정 잠김 문제를 원인별로 나눠서 해결하는 방법을 정리했습니다. Microsoft 계정과 로컬 계정 모두 다루며, 로그인이 아예 안 되는 상황까지 대처법을 안내합니다.

**3줄 요약**
- **비밀번호 틀림** → 잠금 시간(기본 30분) 후 재시도하거나 온라인에서 비밀번호 재설정
- **Microsoft 계정 잠김** → account.live.com에서 본인 인증 후 해제
- **로그인 자체가 불가** → 안전 모드 진입 후 명령 프롬프트로 복구

---

## 윈도우 계정이 잠기는 원인

윈도우 계정 잠김은 크게 세 가지 원인으로 나뉩니다.

### 1. 비밀번호 연속 입력 실패

Windows에는 **계정 잠금 정책**이 있습니다. 비밀번호를 일정 횟수 이상 틀리면 보안을 위해 계정이 자동으로 잠깁니다.

| 설정 항목 | 기본값 | 설명 |
|-----------|--------|------|
| 계정 잠금 임계값 | 10회 | 이 횟수만큼 틀리면 잠김 |
| 계정 잠금 기간 | 30분 | 잠긴 후 자동 해제까지 대기 시간 |
| 잠금 카운터 초기화 | 30분 | 실패 횟수가 0으로 리셋되는 시간 |

> 💡 **참고**: Windows 10/11 가정용 버전에서는 기본적으로 계정 잠금 정책이 비활성화되어 있는 경우가 많습니다. 회사 PC나 도메인에 가입된 PC에서 주로 발생합니다.

### 2. Microsoft 계정 보안 문제

Microsoft 계정(이메일로 로그인하는 계정)은 의심스러운 활동이 감지되면 **원격으로 잠금** 처리됩니다. 다른 지역에서 로그인 시도가 있거나, 비정상적인 메일 발송이 감지되면 계정이 잠길 수 있습니다.

### 3. Windows 업데이트 후 오류

드물지만 Windows 업데이트 이후 PIN이나 비밀번호가 초기화되는 버그가 보고되고 있습니다. 이 경우 올바른 비밀번호를 입력해도 로그인이 안 됩니다.

## Microsoft 계정 잠김 해제 방법

Microsoft 계정(outlook.com, hotmail.com 등)으로 로그인하는 경우, 다른 기기에서 온라인으로 잠금을 해제할 수 있습니다.

### 비밀번호 재설정으로 해제

1. 스마트폰이나 다른 PC에서 [account.live.com/password/reset](https://account.live.com/password/reset) 접속
2. 잠긴 계정의 이메일 주소 입력
3. **본인 확인 방법** 선택 (등록된 이메일, 전화번호, 인증 앱)
4. 인증 코드 입력 후 **새 비밀번호 설정**
5. 잠긴 PC에서 새 비밀번호로 로그인

### 계정 복구 양식 사용

본인 인증 수단에 접근할 수 없는 경우:

1. [account.live.com/acsr](https://account.live.com/acsr) 접속
2. 계정 복구 양식 작성 (최근 비밀번호, 제목, 연락처 등)
3. 제출 후 **24~72시간** 내 결과 확인
4. 하루 최대 2번까지 재제출 가능

> ⚠️ **주의**: 계정 복구 양식은 처리 시간이 걸리므로, 급한 경우 아래 로컬 계정 방법을 먼저 시도하세요.

### 잠금 화면에서 비밀번호 재설정

PC가 인터넷에 연결되어 있다면 잠금 화면에서 직접 재설정할 수도 있습니다.

1. 로그인 화면에서 **"비밀번호를 잊으셨나요?"** 클릭
2. 본인 확인 수단 선택 (이메일/전화)
3. 인증 코드 입력
4. 새 비밀번호 설정 후 로그인

## 로컬 계정으로 잠금 해제하기

Microsoft 계정이 아닌 **로컬 계정**을 사용 중이거나, Microsoft 계정 복구가 어려운 경우 이 방법을 사용합니다.

### 보안 질문으로 비밀번호 재설정

Windows 10/11에서 로컬 계정 설정 시 보안 질문을 등록했다면:

1. 로그인 화면에서 **비밀번호를 틀리면** 나타나는 "암호 재설정" 클릭
2. 보안 질문 3개에 답변 입력
3. 새 비밀번호 설정

### 다른 관리자 계정으로 재설정

PC에 관리자 권한을 가진 다른 계정이 있다면:

1. 다른 관리자 계정으로 로그인
2. **제어판** → **사용자 계정** → **다른 계정 관리**
3. 잠긴 계정 선택 → **암호 변경**

또는 명령 프롬프트(관리자)에서:

```
net user [사용자이름] [새비밀번호]
```

예시: `net user hong 1234` → hong 계정의 비밀번호가 1234로 변경됩니다.

### Microsoft 계정에서 로컬 계정으로 전환

계정 잠김이 반복된다면 **윈도우 로컬계정 변경**을 고려해보세요.

1. **설정** → **계정** → **사용자 정보**
2. **"대신 로컬 계정으로 로그인"** 클릭
3. 현재 Microsoft 계정 비밀번호 입력
4. 로컬 계정 사용자 이름과 비밀번호 설정
5. **로그아웃 후 로컬 계정으로 로그인**

> 💡 **팁**: 로컬 계정으로 전환하면 Microsoft Store, OneDrive 동기화 등 일부 기능이 제한될 수 있습니다. 나중에 다시 Microsoft 계정으로 전환할 수 있으니 부담 없이 시도해보세요.

## 안전 모드에서 계정 복구하기

로그인 화면에서 아무것도 할 수 없는 경우, **안전 모드**로 부팅하면 해결할 수 있습니다.

### 안전 모드 진입 방법

| 방법 | 상황 | 절차 |
|------|------|------|
| 로그인 화면에서 | 로그인 화면까지는 뜨는 경우 | 전원 버튼 → Shift + 다시 시작 |
| 전원 강제 종료 | 로그인 화면도 안 뜨는 경우 | 전원 버튼 10초 길게 눌러 강제 종료 3회 반복 |

**방법 1: 로그인 화면에서**
1. 로그인 화면 우측 하단 **전원 버튼** 클릭
2. **Shift 키를 누른 상태**에서 **다시 시작** 클릭
3. 파란 화면이 나타나면: **문제 해결** → **고급 옵션** → **시작 설정** → **다시 시작**
4. 숫자 **4** 또는 **5**를 눌러 안전 모드 진입

**방법 2: 전원 강제 종료로 복구 환경 진입**
1. PC 전원 버튼을 **10초 이상** 길게 눌러 강제 종료
2. 다시 전원을 켜고 Windows 로고가 보이면 **다시 강제 종료**
3. 이 과정을 **3번 반복**하면 자동 복구 환경(WinRE)이 시작됨
4. **문제 해결** → **고급 옵션** → **시작 설정** → **다시 시작**
5. 숫자 **5**를 눌러 **네트워킹이 있는 안전 모드** 선택

### 안전 모드에서 계정 복구

안전 모드로 부팅되면 **숨겨진 관리자 계정**이나 다른 계정으로 로그인할 수 있습니다.

**명령 프롬프트로 비밀번호 재설정:**

1. 안전 모드에서 로그인 (관리자 계정 또는 다른 계정)
2. **명령 프롬프트**를 **관리자 권한**으로 실행
3. 아래 명령어 입력:

```
net user [사용자이름] [새비밀번호]
```

**숨겨진 관리자 계정 활성화:**

기본 관리자(Administrator) 계정으로 접근이 안 되는 경우:

```
net user Administrator /active:yes
```

이 명령어를 실행하면 로그인 화면에 Administrator 계정이 나타납니다. 이 계정으로 로그인 후 잠긴 계정의 비밀번호를 변경할 수 있습니다.

> ⚠️ **보안 주의**: 복구 후 숨겨진 관리자 계정은 반드시 비활성화하세요.
> ```
> net user Administrator /active:no
> ```

## 로그인이 아예 안 될 때 대처법

위 방법으로도 **윈도우 로그인 안됨** 문제가 해결되지 않는다면 아래 방법을 시도해보세요.

### Windows 설치 USB로 복구

Windows 설치 USB(또는 복구 디스크)가 있다면 가장 확실한 방법입니다.

1. USB를 꽂고 PC 부팅 (BIOS에서 USB 부팅 순서 변경 필요)
2. 언어 선택 후 **"컴퓨터 복구"** 클릭
3. **문제 해결** → **고급 옵션** → **명령 프롬프트**
4. 아래 명령어로 비밀번호 변경:

```
net user [사용자이름] [새비밀번호]
```

5. PC 재부팅 후 새 비밀번호로 로그인

### PIN 재설정

Microsoft 계정 사용자라면 PIN 문제일 수도 있습니다.

1. 로그인 화면에서 **"PIN을 잊어버렸습니다"** 클릭
2. Microsoft 계정 비밀번호 입력
3. 본인 인증 후 **새 PIN 설정**

### 새 로컬 계정 생성

명령 프롬프트에 접근할 수 있다면 새 관리자 계정을 만들 수 있습니다.

```
net user newuser password123 /add
net localgroup administrators newuser /add
```

새 계정으로 로그인한 뒤 기존 계정의 데이터를 복구할 수 있습니다. 기존 계정 데이터는 `C:\Users\[기존사용자이름]` 폴더에 그대로 남아있습니다.

### 최후의 수단: Windows 초기화

**데이터가 삭제**될 수 있으므로 모든 방법을 시도한 후 마지막으로 고려하세요.

1. 로그인 화면 우측 하단 **전원** → **Shift + 다시 시작**
2. **문제 해결** → **이 PC 초기화**
3. **"내 파일 유지"** 선택 (앱은 삭제되지만 개인 파일은 보존)
4. 안내에 따라 초기화 진행

| 옵션 | 개인 파일 | 앱/설정 |
|------|-----------|---------| 
| 내 파일 유지 | 보존 | 삭제 |
| 모든 항목 제거 | 삭제 | 삭제 |

## 계정 잠김 재발 방지 설정

문제를 해결했다면, 같은 일이 반복되지 않도록 예방 설정을 해두세요.

### 1. 비밀번호 복구 수단 등록

- **Microsoft 계정**: 보조 이메일, 전화번호, 인증 앱을 모두 등록
- **로컬 계정**: 보안 질문 3개를 반드시 설정
- **비밀번호 힌트**: 본인만 알 수 있는 힌트 등록

### 2. PIN 설정 (추천)

PIN은 해당 기기에서만 작동하므로 온라인 해킹 위험이 낮습니다.

1. **설정** → **계정** → **로그인 옵션**
2. **PIN(Windows Hello)** → **추가**
3. 4자리 이상 숫자 입력 (문자 포함 가능)

### 3. 계정 잠금 정책 조정 (Pro 이상)

Windows Pro/Enterprise 사용자는 잠금 정책을 직접 조정할 수 있습니다.

1. `Win + R` → `gpedit.msc` 입력
2. **컴퓨터 구성** → **Windows 설정** → **보안 설정** → **계정 정책** → **계정 잠금 정책**
3. **계정 잠금 임계값**을 적절히 설정 (0 = 잠금 없음, 보안상 5~10 추천)
4. **계정 잠금 기간**을 조정 (기본 30분)

> ⚠️ **Home 에디션** 사용자는 `gpedit.msc`를 사용할 수 없습니다. 대신 명령 프롬프트(관리자)에서 아래 명령어로 확인 및 변경할 수 있습니다:
> ```
> net accounts
> ```
> 현재 잠금 정책을 확인할 수 있으며, `net accounts /lockoutthreshold:10` 등으로 변경 가능합니다.

### 4. 비밀번호 관리 습관

- **같은 비밀번호를 여러 곳에 사용하지 않기** (유출 시 연쇄 잠금 위험)
- 비밀번호 관리 프로그램 활용 (브라우저 내장 기능도 가능)
- 중요한 비밀번호는 **오프라인에 메모**해두기

## FAQ

### Q. 비밀번호를 틀린 적 없는데 계정이 잠겼어요

Windows 업데이트 이후 PIN이나 인증 정보가 초기화되는 경우가 있습니다. 또한 Microsoft 계정은 다른 서비스(Outlook, Xbox 등)에서의 비정상 활동이 감지되면 계정 전체가 잠길 수 있습니다. 로그인 화면에서 **"비밀번호를 잊으셨나요?"**를 눌러 재설정하는 것이 가장 빠른 해결법입니다.

### Q. 계정 잠금 시간이 얼마나 되나요?

기본 설정에서 계정 잠금 기간은 **30분**입니다. 30분 후 자동으로 잠금이 해제됩니다. 다만 회사 PC의 경우 IT 관리자가 별도로 설정한 시간이 적용될 수 있으며, 이 경우 관리자에게 문의해야 합니다.

### Q. 로컬 계정 비밀번호도 잊어버렸는데 보안 질문도 설정 안 했어요

이 경우 안전 모드에서 숨겨진 관리자 계정을 활성화(`net user Administrator /active:yes`)하여 비밀번호를 재설정하는 방법이 있습니다. 관리자 계정도 잠겨있다면 **Windows 설치 USB**를 사용하거나, 최후의 수단으로 "이 PC 초기화"의 "내 파일 유지" 옵션을 사용할 수 있습니다.

### Q. 회사 PC(도메인 가입)에서 계정이 잠겼어요

도메인에 가입된 PC는 **Active Directory** 정책에 의해 관리됩니다. 이 경우 본인이 직접 해제할 수 없으며, **IT 관리자**에게 연락하여 계정 잠금 해제를 요청해야 합니다. 일반적으로 관리자가 "Active Directory 사용자 및 컴퓨터"에서 계정 잠금을 해제해줍니다.

### Q. 윈도우 로그아웃은 어떻게 하나요?

로그아웃은 **시작 메뉴** → **사용자 아이콘** → **로그아웃**을 클릭하면 됩니다. 또는 `Ctrl + Alt + Delete`를 누른 후 **"로그아웃"**을 선택할 수도 있습니다. 다른 계정으로 전환해야 할 때 유용합니다.
