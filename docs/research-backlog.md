# 키워드 조사 백로그

발행 측정 사이클(~2주) 동안 다음 라운드 조사·큐레이션을 병렬 진행하기 위한 추적 문서. 측정 결과가 나오면 우선순위 재배열.

## 타겟 페르소나

**40+ 건강 관심층 (한국어 네이버 검색 사용자)**. 자세한 기준 — `docs/content-strategy.md` 의 "타겟 페르소나" 섹션 참조.

이 페르소나에 맞지 않는 키워드는 후보 단계에서 거절.

## 진행 상태 분류

- 🟢 **진행 중** — 데이터 받아서 큐레이션·자료조사 단계
- 🟡 **측정 대기** — 발행했고 결과 나오면 다음 발행 결정
- 🔵 **후보** — 데이터 받기 전 기획 단계
- ⚪ **보류** — 측정 결과 의존, 페르소나 부합 재검토 필요
- ❌ **거절** — 안티 패턴 / 페르소나 불일치 / 폐기 가설

## ❌ 거절 (썸원, 2026-06-28)

### 썸원 질문 모음 / 속마음편지 (시드: 썸원, 클러스터 2) — 폐기

- 클러스터 1(반려몽 종류·키우는 법)은 2026-06-27 발행 완료 — `sumone-pet-types`, `sumone-pet-growth-guide`
- 클러스터 2 1차 보류 사유: 질문 모음 실제 예시 못 모음, 속마음편지=느림보편지 여부 미확인, 라이크 유는 SERP 무관 영역 점유로 확정 보류
- 재조사 결과: 공식 FAQ(sumone.co/faq) 51개 항목 확보 → 속마음편지는 근거 없는 가설로 폐기, 대신 **썸원타임 변경 / 질문이 안와요·밀렸어요 / 커플연결 끊기·복구** 3개로 후보 교체. SERP 1면 공실(개인 블로그 0건) 확인까지 마쳤음
- **최종 폐기 사유**: 사용자가 키워드 도구로 검색량·문서수 직접 조회 → 수치 매력 없음 확인. SERP 공실이라고 검색 수요가 있다는 뜻은 아니었음
- **메타 학습**: SERP 1면 점유 현황만으로 발행 결정하지 말 것. 키워드 도구 수치(검색량·문서수)를 가능한 먼저 확인 — `content-strategy.md` 예외 조항 업데이트 반영

## ❌ 거절 (허리.json 큐레이션 — 2026-05-26)

데이터 파일: `data/keywords/back-pain-2026-05-26.json` (95 키워드). 메인 + 독립 심사 병렬 큐레이션 결과 **교집합 1편(`허리 디스크 터지면`)** 만 시도 가치였고, 그것도 "도박" 수준. 사용자가 폐기 결정 (이유: 허리 영역 의료·한방 권위 점유율 압도적, 1편 도박을 위해 도메인 신뢰도 희석 위험).

| 거절 군 | 사유 |
|---|---|
| 의류 수선 (바지/청바지/슬랙스/치마/레이어드/수선) — 18% | 40+ 건강 페르소나 이탈 |
| 의료기관 점유 (병원/도수치료/주사/수술/견인치료/내시경 감압술) | 외부 도메인 진입 불가 |
| 헤드 키워드 (문서 10만+) — 허리통증/허리디스크/협착증/근육통/운동/마사지/베개/디스크 부위별 통증 등 | 진입 거의 불가 |
| 의료용 보호대·견인기·교정기 — 어필리에이트 농장 + 쇼핑몰 점유 | 진입 슬롯 0 + 안티 패턴(추천 TOP N) 위반 |
| 낚시 구명조끼 (다이와/구명벨트) | 페르소나 이탈 |
| 자기복제 — 허리 삐끗·담·허리베개 | `lower-back-strain-vs-cramp`, `jeong-seon-geun-back-pillow` 이미 점유 |

**보류 1편**: `허리 디스크 터지면` (3,920) — 향후 도메인 권위 회복 시 재고. Pattern 5 불안감 진단형으로 의료기관 점유 영역 우회 가능성은 있음 (단 현재는 도박).

**메타 학습**: 허리 영역은 의료·한방 권위가 모든 건강 분야 중 가장 강함. JSON 데이터 "쉬움" 라벨이 다수 매겨졌어도 SERP 1면은 의료기관·어필리에이트 농장으로 도배. **데이터 라벨 ≠ 실제 진입 가능성** 원칙 재확인.

## ❌ 거절 (비엔알 시드 — 2026-06-28)

데이터: 사용자 제공 표 4개 키워드(비엔알17유산균 4,590 / 비엔알17 1,500 / 비엔알씬 530 / 비엔알17유산균가격 415, 전부 "쉬움"·기회지수 75+). 메인 + 독립 심사 병렬 SERP 실측 결과 **교집합 0편** — 4개 전부 거절.

| 키워드 | 사유 |
|---|---|
| 비엔알17 / 비엔알17유산균 | SERP 1면이 11번가·G마켓·올리브영·코스트코·다나와·홈쇼핑모아 + 브랜드 공식몰(비에날몰)로 완전 포화, 개인 블로그 0 |
| 비엔알씬 | 제품 브랜드명 자체 — "브랜드 소개 일반론" 안티 패턴 정면 해당 |
| 비엔알17유산균가격 | "가격" 단일답변형처럼 보이나 옵션·변동가 多 — 쿠차·다나와 가격비교 사이트가 이미 그 의도 충족 |
| (검토했으나 폐기) 성분(비엔알17) vs 제품명(비에날씬) 차이 설명 앵글 | pillyze.com이 라인업별 효능·부작용·후기·복용법까지 이미 선점 — 빈 자리 아님 |

**메타 학습**: 노출광고수(7~8개)가 핵심 신호였음. 광고주가 많다는 건 그 키워드가 쇼핑/구매 의도이고 이미 쇼핑몰·가격비교 사이트가 정확히 그 의도를 충족 중이라는 뜻 — "쉬움" + 기회지수 80대 라벨에도 불구하고 정보형 개인 블로그가 비집을 자리가 없었다. 건강기능식품 중 **브랜드 공식몰이 존재하는 단일 제품/성분** 키워드는 의료기관·공공기관만큼이나 진입이 막혀있다는 패턴으로 기록.

**최종 결정 (사용자 오버라이드)**: 거절 권고에도 사용자가 "그래도 시도해보자"로 진행 결정. 단, 4편 도박 대신 **1편(허브)으로 압축**해 노출을 최소화 — `비엔알17(균주명) vs 비에날씬(제품 브랜드명)` 구분 + 식약처 인증 범위 정직하게 짚기 + 가격 FAQ 흡수. 슬러그 `bnr17-vs-bnrthin`, 2026-06-28 발행. 차별화 포인트: 경쟁사(pillyze.com)의 "효능·부작용·후기 나열형" 포맷과 달리 "식약처 인증이 보장하는 것과 보장 안 하는 것"을 구분하는 정직한 정보형 톤. 쿠팡 링크는 의도적으로 생략(단일 브랜드 추천이 "브랜드보다 성분표 확인" 메시지와 충돌). 측정 사이클에서 노출·클릭 추적 필요 — SERP 포화 상태에서 개인 블로그가 비집을 수 있는지의 실험 표본.
